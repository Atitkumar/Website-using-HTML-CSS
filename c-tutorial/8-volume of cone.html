Program to find the volume of the cone using c.

Formula - volume of cone = pie * r2 * h/3
#include <stdio.h>    
   int main()   
 {                    
   int height=28;  
    int radius=15;  
    double pie=3.14285714; 
    double volume = pie * (radius * radius) * height/3;  
    printf("Volume of the cone is = %f ", volume); 
    return 0;
 }

Output:-

Volume of the cone is = 6599.999994
