Program to determine whether a given number is a happy number using c.

The happy number can be defined as a number which will yield 1 when it is replaced by the sum of the square of its digits repeatedly. If this process results in an endless cycle of numbers containing 4, then the number is called an unhappy number.

32 is a happy number as the process yields 1 as follows

32 + 22 = 13
12 + 32 = 10
12 + 02 = 1

Some of the other examples of happy numbers are 7, 28, 100, 320 and so on. The unhappy number will result in a cycle of 4, 16, 37, 58, 89, 145, 42, 20, 4, ....
#include <stdio.h>      
// HappyNumber() will determine whether a number is happy or not. 
int HappyNumber(int numb){  
int remi = 0, sum = 0;  
//Calculates the sum of squares of digits.  
while(numb > 0){  
  remi = numb%10;  
  sum = sum + (remi*remi);  
  numb = numb/10;  
 }  
  return sum;  
}        
int main()  
{  
 int numb = 82;  
 int result = numb;  
 while(result != 1 && result != 4){  
   result = HappyNumber(result);  
}  
//Happy number always ends with 1  
 if(result == 1)  
   printf("%d is a happy number", numb);  
//Unhappy number ends in a cycle of repeating numbers which contains 4  
 else if(result == 4)  
  printf("%d is not a happy number", numb); 
 return 0;  
}

Output:-

82 is a happy number
