Program to calculate area of rectangle using c

Formula - Area of Rectangle = width * Height

#include <stdio.h>
int main()
{
int width_ = 3;
int height_ = 7;
int AreaOfRectangle = width_ * height_;
printf("Area of the rectangle is = %d",AreaOfRectangle);
}

Output:-

Area of the rectangle is=21
