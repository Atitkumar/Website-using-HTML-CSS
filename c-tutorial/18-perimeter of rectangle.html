Program to find the perimeter of the rectangle using c.

Formula - Suppose a, b, c, d are the four sides of rectangle.
a = c and b = d
Perimeter Of Rectangle = a + b + c + d = a + b + a + b = 2 (a + b).
#include <stdio.h>    
int main()   
{                    
 float aa , bb, cc, dd ,perimeters;  
 aa = cc = 7;  
 bb = dd = 14;  
 perimeters = 2 * (aa + bb);  
 printf("Perimeter of Rectangle is :- %0.1f",perimeters);  
 return (0);    
}

Output:-

Perimeter of Rectangle is :- 42.0
